# Australian_rainfall
This project was for a Data Science Class and contains a wide range of analysis. This project was originally done in Dec 2024. 

** Requirements **
This code using a python kernal and is dependet on the following libraries: pandas, numpy, seaborn, matplotlib.pyplot, os, and sklearn. 

Please visit the following website to source the data if you want to run your own analysis on updated data: https://reg.bom.gov.au/climate/data/
